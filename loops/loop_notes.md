Loop Notes (Python)

What is a loop?
A loop is a programming structure that repeats a block of code as long as a certain condition is true. It helps us to avoid writing the same code multiple times.

Difference between for and while
-for loop: Used when we know how many times we want to repeat the code. It iterates over a sequence (like a list, tuple, or range).


-while loop: Used when we don't know the exact number of iterations. It runs as long as a given condition is True.

What does break do?
The break statement immediately stops the loop entirely, even if the loop condition is still True. The program then continues with the next line of code after the loop.

What does continue do?
The continue statement skips the rest of the code inside the current iteration and jumps to the next iteration of the loop. It does not stop the loop; it just skips one step.

What is a nested loop?
A nested loop is a loop inside another loop. The "inner loop" will complete all its iterations for each single iteration of the "outer loop". It is useful for working with multi-dimensional data (like matrices) or generating patterns (like multiplication tables).
